# Renal Cyst Segmentation using U-Net with ResNet50 and Explainable AI

## Overview
This project focuses on automatic renal cyst segmentation from kidney medical images using a U-Net architecture with a ResNet50 backbone. Explainable AI (XAI) techniques are incorporated to improve model interpretability and transparency.

## Features
- Automated renal cyst segmentation
- U-Net with ResNet50 encoder
- Medical image preprocessing
- Explainable AI visualization
- Performance evaluation metrics

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Explainable AI (XAI)

## Applications
- Medical image analysis
- Kidney disease diagnosis support
- Clinical decision assistance
